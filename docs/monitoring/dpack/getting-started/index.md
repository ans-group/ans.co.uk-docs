# Getting started with Live Optics (formerly DPACK)

You can request a Live Optics (formerly DPACK) capture by contacting your ANS Account Manager or raising a support ticket through [ANS Glass](https://ans.glass/support-and-help/incidents).

Running a Live Optics capture will allow you and ANS to measure a wide variety of metrics such as:

- CPU & RAM usage
- IOPS
- IO size
- Disk latency & throughput
- Queue depth
- Network throughout
- Hard page faults per second.

A Live Optics can be run on any [Windows](../windows/) or [Linux](../linux/) based operating system.
