# [WooCommerce](https://woocommerce.com/)

WooCommerce is an open-source e-commerce plugin for WordPress. The core components of our optimised WooCommerce stack are:

- **Firewall**: [Dedicated/Shared Firewall](../../network/firewalls/)
- **Web service**: [NGINX](../../operating-systems/linux/nginx/)
- **PHP utilities**: [PHP](./php/) , [PHP-FPM](../../operating-systems/linux/php-fpm/), [OPCache](./opcache/)
- **Mail services**: [Postfix](../../email/postfix/)
- **SSH/FTP access**: [SSH](../../operating-systems/linux/ssh/), [VSFTPd](../../operating-systems/linux/ftp/setup-vsftpd/)
- **Database services**: [Percona](../../operating-systems/linux/percona/), [Redis](./redis/)
- **Version control utilities**: [Git](../../operating-systems/linux/git/)
- **File system distribution**: [NFS](../../operating-systems/linux/nfs/)

We have a number of guides for our WooCommerce optimised stack:

- [Adding domain(s) to the stack](./adding-domains-to-the-stack/)
- [PHP](./php/)
- [OPCache](./opcache/)
- [Redis](./redis/)
