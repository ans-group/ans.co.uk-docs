# [AeroCommerce](https://www.aerocommerce.com/)

_AeroCommerce_ is a feature-rich ecommerce solution with unlimited customisation and extensibility. The core components of our optimised _AeroCommerce_ stack are:

- **Firewall**: [Dedicated/Shared Firewall](../../network/firewalls/)
- **Web service**: [NGINX](../../operating-systems/linux/nginx/)
- **PHP utilities**: [PHP](./php/) , [PHP-FPM](../../operating-systems/linux/php-fpm/)
- **Mail services**: [Postfix](../../email/postfix/)
- **SSH/FTP access**: [SSH](../../operating-systems/linux/ssh/), [VSFTPd](../../operating-systems/linux/ftp/setup-vsftpd/)
- **Database services**: [Percona](../../operating-systems/linux/percona/)
- **Version control utilities**: [Git](../../operating-systems/linux/git/)
- **File system distribution**: [NFS](../../operating-systems/linux/nfs/)

We have a number of guides for our AeroCommerce optimised stack:

- [Adding domain(s) to the stack](./adding-domains-to-the-stack/)
- [PHP](./php/)

You can find out more about our offerings for these stacks [here](https://www.ans.co.uk/cloud-services/managed-hosting/e-commerce-hosting/).
