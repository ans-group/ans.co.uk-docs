# [Magento 2](https://business.adobe.com/products/commerce.html)

In 2015 we created our Magento 2 optimised stack. The core components of this stack are:

- **Firewall**: [Dedicated/Shared Firewall](../../../network/firewalls/)
- **Web Service**: [NGINX](../../../operating-systems/linux/nginx/)
- **PHP utilities**: [PHP](./php/), [PHP-FPM](../../../operating-systems/linux/php-fpm/), [OPCache](./opcache/), [Composer](./composer/)
- **Mail services**: [Postfix](../../../email/postfix/)
- **SSH/FTP access**: [SSH](../../../operating-systems/linux/ssh/), [VSFTPd](../../../operating-systems/linux/ftp/setup-vsftpd/)
- **HTTP caching utilities**: [Varnish](./varnish/)
- **Database services**: [Percona](../../../operating-systems/linux/percona/), [Elasticsearch](../../../operating-systems/linux/elasticsearch/), [Redis](./redis/)
- **Version control utilities**: [Git](../../../operating-systems/linux/git/)
- **File system distribution**: [NFS](../../../operating-systems/linux/nfs/)
