# ANS-supported VPNs

There are three different types of VPNs we support if your servers are behind an ANS dedicated firewall:

- **Site to Site VPN:** This type of VPN is mainly used for connecting between two physical locations e.g. your office to a ANS data centre. You will need a firewall/router to support IPsec site to site VPN at your office/other location.

- **Traditional IPsec remote access VPN:** This VPN is used for connecting a mobile device to your ANS solution behind a dedicated firewall. It will support Microsoft Windows XP / 7 (32bit/64bit) / Linux / MacOS.

- **WebSSL VPN:** This VPN is used for connecting mobile devices to your ANS solution behind a dedicated firewall. It will support Microsoft Windows XP / 7 (32bit/64bit)/ 10 / Linux / MacOS.

If your servers are behind ANS shared firewalls, we will not support the creation of VPNs. However we will be able to open ports for customers who run VPN termination on your servers.
