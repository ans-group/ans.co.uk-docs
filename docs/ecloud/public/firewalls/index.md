# Firewalls

Here you’ll find all the guidance relating to ANS dedicated and shared firewalls, as well as VPNs. ANS offers two primary types of firewalls, shared and dedicated. Both offer the basic functions you would expect from a firewall such as controlling ports, locking down services by IP, and the ability to blocking IPs.

Our dedicated firewalls allow much finer control, as well as advanced features like VPNs, and the ability to allow ICMP traffic. You can also have more advanced setups, including redundant pairs of firewalls.
